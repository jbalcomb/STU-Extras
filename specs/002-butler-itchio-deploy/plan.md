# Implementation Plan: Butler itch.io Deployment

**Branch**: `002-butler-itchio-deploy` | **Date**: 2026-03-08 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/002-butler-itchio-deploy/spec.md`

## Summary

Add shell scripts that wrap itch.io's Butler CLI tool to deploy WASM builds to itch.io with a single command. Includes deploy-only, build-and-deploy, and configurable project target. This is external tooling (shell scripts) — no changes to the C++ application code, data layer, or packed structs.

## Technical Context

**Language/Version**: Bash (cross-platform shell scripts, also usable on Windows via Git Bash / MSYS2)
**Primary Dependencies**: Butler (itch.io CLI tool, installed separately by developer), Emscripten SDK (for WASM builds)
**Storage**: Configuration file (`deploy.conf`) in repository root
**Testing**: Manual validation — shell scripts verified by dry-run mode and actual deployment
**Target Platform**: Windows (primary development), Linux/macOS (secondary)
**Project Type**: Build tooling / shell scripts (not compiled, not part of the application)
**Performance Goals**: Deployment completes in under 2 minutes (excluding build time)
**Constraints**: Butler must be pre-installed and authenticated; Emscripten SDK must be available for build-and-deploy mode
